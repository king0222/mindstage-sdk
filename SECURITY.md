# Security Policy

## Supported Versions

We only provide security updates for the latest version of the SDK.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it to us as soon as possible.

You can report vulnerabilities by:

- Opening a private issue on GitHub (if supported)
- Contacting us directly through [MindStage](https://mindstage.app)

Please include as much detail as possible, including steps to reproduce the issue and any potential impact. We will investigate and address the issue as quickly as possible.
